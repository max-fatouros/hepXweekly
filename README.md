# HEP X Weekly


## Installation
```
pip install git+git@github.com:ShreyasBakare/hepXweekly.git
```

## Usage
Everytime you run `hepxweekly` you will get the latest papers from arxiv that you can further filter using keywords/ read title, author, abstract or enter the sr. no. to directly open the pdf of that paper in your browser.
